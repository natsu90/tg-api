1. Create a new NodeJS application on Openshift with following Git URL, https://github.com/natsu90/tg-api.git, or click here
  2. SSH to your OpenShift application; ssh your-application-id@your-application-url e.g ssh 565fcf2c89f57576490001c1@telegram-namialus.rhcloud.com
  3. Log into Telegram CLI for first time, $TELEGRAM_HOME/bin/telegram-cli -k $TELEGRAM_HOME/tg-server.pub
  4. While in SSH session, set your environment variables of WEBHOOK and API_KEY (default to OPENSHIFT_SECRET_TOKEN); echo "http://requestb.in/blablabla" > $OPENSHIFT_HOMEDIR/.env/user_vars/WEBHOOK
  5. Restart application from Openshift account
